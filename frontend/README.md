# DNA musical explorer - Frontend

Interface limpa e responsiva, com foco em usabilidade, intuitividade e experiência agradável ao usuário.


##  Tecnologias

- **React**
- **Next.js**
- **Tailwind CSS**
- **CSS**
- **HTML**

---

##  Conceito de Interface

O frontend foi desenvolvido com foco em:
- Clareza visual e layout limpo
- Responsividade para diferentes tamanhos de tela
- Navegação simples e intuitiva
- Boa experiência do usuário ao explorar músicas e resultados
